Task: Install a stateful service (MongoDB) on Kubernetes using Helm.

Technologies used: Kuberenetes, Helm, MongoDB, Mongo Express, Linode LKE and Linux.

Project Description: 
1. Create a managed K8s cluster with Linode Kuberenetes Engine.
2. Deploy replicated MongoDB service in LKE cluster using a Helm chart.
3. Configure data persistance for MongoDB with Linode's cloud storage. 
4. Deploy UI client Mongo Express for MongoDB.
5. Deploy and configure nginx ingress to access the UI application from browser.
